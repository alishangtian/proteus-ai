# Stock Research Skill

## Overview
A professional stock research and analysis skill for generating comprehensive stock research reports, investment recommendations, and trading guidance.

## Skill Structure

### Main Files
- `SKILL.md` - Main skill documentation with complete analysis framework
- `README.md` - This file
- `examples/` - Example files and templates
  - `stock_analysis_example.py` - Python stock analysis example
  - `data_collection_template.md` - Data collection template
  - `research_report_template.md` - Research report template

## When to Use This Skill

Use this skill when:
- Analyzing specific stocks or companies
- Generating investment research reports
- Providing buy/sell/hold recommendations
- Creating trading strategies and position management guidance
- Conducting due diligence on investment opportunities

## Core Analysis Framework

The skill follows a 5-step framework:

1. **Data Collection** - Gather comprehensive data from multiple sources
2. **Fundamental Analysis** - Evaluate financial health, valuation, and growth
3. **Technical Analysis** - Analyze price patterns, trends, and momentum
4. **Qualitative Assessment** - Consider competitive position, management, risks
5. **Synthesis & Recommendation** - Generate actionable investment guidance

### Professional Reporting

## Data Requirements

For effective analysis, gather:
- Company financial statements (minimum 3 years)
- Stock price history
- Industry comparison data
- Analyst estimates and ratings
- Relevant economic indicators

## Integration with Other Skills

This skill works well with:
- **Manus skill** - For complex, multi-step research projects
- **Web scraping tools** - For data collection from financial websites
- **Data visualization tools** - For creating charts and graphs
- **Document generation tools** - For formatting professional reports

## Important Notes

### Limitations
- Relies on publicly available data which may be incomplete or delayed
- Cannot provide real-time market data without appropriate APIs
- Analysis is informational only, not personalized investment advice

### Disclaimers Required
Always include appropriate disclaimers when providing stock analysis:
> "This analysis is for informational purposes only and should not be construed as investment advice. Please conduct your own due diligence and consult with a qualified financial advisor."

## Quick Start

1. **Identify the stock** to analyze
2. **Gather data** using the provided templates
3. **Apply the analysis framework** from SKILL.md
4. **Generate a report** using the template
5. **Provide recommendations** with clear rationale and risk disclosures
