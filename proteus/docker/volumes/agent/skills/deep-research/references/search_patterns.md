# Search Patterns Guide

Effective search strategies for research.

## Basic Search Operators

1. **Phrase Search**: "exact phrase"
2. **OR Search**: term1 OR term2
3. **Site Restriction**: site:example.com
4. **File Type**: filetype:pdf
5. **Title Search**: intitle:keyword

## Search Strategy

1. Start broad to understand landscape
2. Refine based on initial findings
3. Use multiple sources
4. Verify critical information

## Example Queries

- "market analysis" 2024 site:.gov
- "technology trends" filetype:pdf
- "industry report" OR "market research"
