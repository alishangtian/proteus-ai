# 信息来源评估指南

## 概述
信息来源评估是深度研究的基础，确保研究建立在可信信息之上。

## 信息来源分类体系

### Tier 1: 权威来源 (可信度评分: 9-10)
**特点**: 最高可信度，经过严格审核

**包括**:
- 学术期刊论文（同行评审）
- 官方统计数据（政府机构）
- 专利文档
- 国际组织报告（UN, WHO, IMF等）
- 法院判决和法律文件

**评估要点**:
- 出版机构声誉
- 同行评审过程
- 方法论透明度
- 数据可验证性

### Tier 2: 专业来源 (可信度评分: 7-8)
**特点**: 专业可信，有一定审核机制

**包括**:
- 行业研究报告（Gartner, McKinsey, BCG等）
- 专业机构白皮书
- 专家访谈和演讲
- 技术标准文档
- 上市公司年报和财报

**评估要点**:
- 作者/机构专业背景
- 方法论描述
- 利益冲突声明
- 数据来源说明

### Tier 3: 媒体和分析来源 (可信度评分: 5-6)
**特点**: 有一定可信度，但可能有偏见

**包括**:
- 主流媒体报道（Reuters, Bloomberg, WSJ等）
- 行业分析文章
- 专业博客和专栏
- 公司新闻稿和公告
- 政府新闻发布

**评估要点**:
- 媒体机构声誉
- 记者专业背景
- 事实核查机制
- 引用来源质量

### Tier 4: 社区和社交来源 (可信度评分: 1-4)
**特点**: 可信度较低，需谨慎验证

**包括**:
- 社交媒体内容
- 论坛讨论
- 个人博客
- 用户评论和评价
- 维基百科（作为起点而非最终来源）

**评估要点**:
- 作者可信度
- 一致性检查
- 可验证性
- 偏见识别

## 评估维度和指标

### 1. 权威性 (权重: 30%)
**评估**: 来源发布机构或作者的专业地位

**指标**:
- 机构/作者在领域的专业声誉
- 相关资质和认证
- 历史准确性和可靠性记录
- 同行认可程度

### 2. 准确性 (权重: 25%)
**评估**: 信息的准确和精确程度

**指标**:
- 事实核查结果
- 数据可验证性
- 错误和更正历史
- 与其他可靠来源的一致性

### 3. 时效性 (权重: 15%)
**评估**: 信息的时效性和相关性

**指标**:
- 发布日期
- 信息更新频率
- 对快速变化领域的适用性
- 是否有更近期的更新

### 4. 全面性 (权重: 10%)
**评估**: 信息的完整和全面程度

**指标**:
- 覆盖范围的广度
- 细节深度
- 不同观点的包含
- 局限性说明

### 5. 客观性 (权重: 20%)
**评估**: 信息的客观和公正程度

**指标**:
- 利益冲突声明
- 偏见和倾向性
- 平衡性（正反观点）
- 宣传或营销内容比例

## 评估流程

### 步骤1: 初步分类
根据来源类型进行初步Tier分类。

### 步骤2: 详细评估
按五个维度进行详细评分。

### 步骤3: 交叉验证
与其他来源进行比较验证。

### 步骤4: 综合评分
计算综合可信度评分。

### 步骤5: 记录存档
记录评估结果和理由。

## 可信度评分公式

```
综合评分 = (权威性×0.30 + 准确性×0.25 + 时效性×0.15 + 全面性×0.10 + 客观性×0.20)
Tier调整 = 根据Tier类别调整最终评分
```

## 常见陷阱与避免方法

### 陷阱1: 权威偏见
**问题**: 过度依赖权威来源，忽视其他有价值信息
**避免**: 平衡使用不同Tier的来源，关注信息质量而非仅仅权威性

### 陷阱2: 确认偏误
**问题**: 只选择支持预设观点的来源
**避免**: 主动寻找反对观点和证据

### 陷阱3: 时效性忽视
**问题**: 使用过时信息
**避免**: 检查发布日期，优先使用最近2年内的信息

### 陷阱4: 表面评估
**问题**: 仅凭来源类型判断可信度
**避免**: 进行详细的多维度评估

## 特殊来源处理

### 人工智能生成内容
**处理原则**:
1. 视为起点而非最终来源
2. 必须验证关键事实
3. 检查信息来源引用
4. 评估逻辑一致性

### 社交媒体内容
**处理原则**:
1. 谨慎对待，必须验证
2. 考虑作者背景和动机
3. 检查是否被其他可靠来源引用
4. 关注趋势而非单一观点

### 匿名来源
**处理原则**:
1. 提高验证标准
2. 寻找其他来源佐证
3. 考虑信息提供动机
4. 明确标注匿名性质

## 质量保证检查清单

- [ ] 每个关键事实有至少2个独立来源支持
- [ ] Tier 1和Tier 2来源比例 ≥ 60%
- [ ] 信息来源时间跨度合理（最新信息包含在内）
- [ ] 不同观点和立场都有代表
- [ ] 所有来源都经过评估和记录
- [ ] 低可信度来源已明确标注并谨慎使用
