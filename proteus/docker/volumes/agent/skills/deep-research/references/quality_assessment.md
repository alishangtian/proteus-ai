# 研究质量评估指南

## 概述
质量评估是深度研究的核心环节，确保研究成果达到专业标准。

## 质量维度

### 1. 问题清晰度 (权重: 15%)
**评估标准**: 研究问题是否明确、具体、可操作

**检查清单**:
- [ ] 研究问题用清晰的语言表述
- [ ] 问题范围明确界定
- [ ] 关键术语有明确定义
- [ ] 研究目标具体可衡量
- [ ] 排除范围明确说明

**评分标准**:
- 优秀 (13-15分): 问题极其清晰，边界明确
- 良好 (10-12分): 问题基本清晰，稍有模糊
- 合格 (7-9分): 问题表述一般，需要澄清
- 需改进 (<7分): 问题模糊，边界不清

### 2. 方法严谨性 (权重: 20%)
**评估标准**: 研究方法是否科学、合理、透明

**检查清单**:
- [ ] 研究方法适合研究问题
- [ ] 数据收集方法合理
- [ ] 样本选择有代表性
- [ ] 分析技术适当
- [ ] 方法局限性被承认

**评分标准**:
- 优秀 (18-20分): 方法极佳，完全透明
- 良好 (15-17分): 方法合理，基本透明
- 合格 (12-14分): 方法一般，部分透明
- 需改进 (<12分): 方法不当，不透明

### 3. 信息全面性 (权重: 20%)
**评估标准**: 是否覆盖关键信息来源和不同观点

**检查清单**:
- [ ] 包含主要信息源类型
- [ ] 覆盖不同观点和立场
- [ ] 包含最新信息
- [ ] 关键数据点有多个来源
- [ ] 信息深度足够

**量化指标**:
- Tier 1来源: ≥3个
- 总来源数: ≥8个
- 观点多样性: ≥3种不同立场
- 时间跨度: 包含最近6个月的信息

**评分标准**:
- 优秀 (18-20分): 极其全面，多源验证
- 良好 (15-17分): 比较全面，主要来源覆盖
- 合格 (12-14分): 基本全面，关键信息覆盖
- 需改进 (<12分): 信息片面，关键来源缺失

### 4. 分析深度 (权重: 25%)
**评估标准**: 分析是否深入，是否挖掘到根本原因

**检查清单**:
- [ ] 超越表面现象
- [ ] 识别根本原因
- [ ] 考虑多因素相互作用
- [ ] 进行趋势预测
- [ ] 提供独特洞察

**深度指标**:
- 因果链长度: ≥3层
- 影响因素: ≥5个关键因素
- 场景分析: ≥2个不同场景
- 敏感性分析: 关键变量变化影响评估

**评分标准**:
- 优秀 (23-25分): 深度极佳，根本原因清晰
- 良好 (20-22分): 深度良好，多因素分析
- 合格 (17-19分): 基本深度，表面分析
- 需改进 (<17分): 深度不足，停留在表面

### 5. 建议可行性 (权重: 15%)
**评估标准**: 建议是否具体、可操作、可实现

**检查清单**:
- [ ] 建议具体明确
- [ ] 有实施步骤
- [ ] 考虑资源约束
- [ ] 有优先级排序
- [ ] 有风险评估

**可行性指标**:
- 具体行动: ≥3个具体行动
- 时间框架: 有明确时间安排
- 资源需求: 资源需求明确
- 成功指标: 有可衡量的成功标准

**评分标准**:
- 优秀 (13-15分): 建议极佳，完全可操作
- 良好 (10-12分): 建议良好，基本可操作
- 合格 (7-9分): 建议一般，需要细化
- 需改进 (<7分): 建议模糊，不可操作

### 6. 呈现质量 (权重: 5%)
**评估标准**: 报告是否清晰、易读、专业

**检查清单**:
- [ ] 结构清晰逻辑
- [ ] 语言简洁明了
- [ ] 图表适当有效
- [ ] 格式一致专业
- [ ] 无明显错误

**评分标准**:
- 优秀 (5分): 呈现极佳，完全专业
- 良好 (4分): 呈现良好，基本专业
- 合格 (3分): 呈现一般，需要改进
- 需改进 (<3分): 呈现差，影响理解

## 总体评分与等级

### 总分计算
```
总分 = Σ(维度得分 × 权重)
```

### 质量等级
- **卓越 (90-100分)**: 可直接用于战略决策
- **优秀 (80-89分)**: 轻微优化后即可使用
- **良好 (70-79分)**: 需要中等程度的完善
- **合格 (60-69分)**: 需要显著改进
- **需改进 (<60分)**: 需要重新研究

## 质量保证流程

### 阶段1: 研究设计阶段
1. 明确质量要求
2. 制定质量检查点
3. 确定评估标准

### 阶段2: 研究执行阶段
1. 定期质量自检
2. 关键节点质量评审
3. 调整研究方法

### 阶段3: 报告完成阶段
1. 全面质量评估
2. 同行评审（可选）
3. 用户反馈收集

### 阶段4: 持续改进
1. 质量评估结果分析
2. 方法论改进
3. 模板和工具优化

## 常见质量问题与改进

### 问题1: 信息片面
**表现**: 只收集支持预设观点的信息
**改进**: 主动寻找反面证据，指定"魔鬼代言人"角色

### 问题2: 分析表面
**表现**: 停留在表面现象，未挖掘根本原因
**改进**: 应用"5个为什么"技术，深入分析因果链

### 问题3: 建议空泛
**表现**: 建议过于笼统，缺乏具体行动
**改进**: 使用SMART原则制定具体建议

### 问题4: 方法不当
**表现**: 研究方法不适合研究问题
**改进**: 研究前进行方法论评审，选择合适的分析框架
