# 🚀 Deep Research 快速入门指南 v6.0

## 五分钟掌握深度研究

### 1. 核心概念速览

**双模式研究**：
- **深度推理模式**：单智能体深度探索，适合复杂问题
- **并行执行模式**：多智能体并行，适合大规模比较

**五层质量保证**：
1. 过程监控 2. 源评估 3. 矛盾调解 4. 反思修订 5. 最终审查

**模板系统**：
- 🎓 academic.md - 学术研究
- 🎯 executive.md - 高管决策  
- 🔬 technical.md - 技术分析
- ⚡ quick.md - 快速决策
- 📄 standard.md - 标准报告

### 2. 立即开始研究

#### 步骤1: 定义研究问题
```python
研究主题 = "你的研究主题"
研究目标 = [
    "目标1: 分析...",
    "目标2: 评估...", 
    "目标3: 建议..."
]
研究模式 = "自动选择"  # 或手动指定"深度推理"/"并行执行"
输出模板 = "standard.md"  # 默认模板
```

#### 步骤2: 执行搜索策略
```python
# 搜索策略示例
搜索关键词 = [
    "核心关键词 2025 趋势",
    "核心关键词 技术 最新进展", 
    "核心关键词 市场 规模",
    "核心关键词 挑战 风险"
]
搜索轮次 = 3  # 推荐3轮迭代搜索
```

#### 步骤3: 应用分析框架
根据研究类型选择分析框架：
- **市场分析** → PESTEL+、波特五力+
- **技术评估** → 技术采纳生命周期2.0  
- **战略规划** → SWOT 2.0、BCG矩阵
- **风险评估** → 风险矩阵、情景分析

#### 步骤4: 生成研究报告
```python
# 使用内置模板
模板选择规则:
  - 学术研究 → academic.md
  - 技术决策 → technical.md
  - 高管决策 → executive.md
  - 紧急决策 → quick.md
  - 默认选择 → standard.md
```

### 3. 搜索技巧速成

#### 关键词优化
```python
# 从基础关键词扩展
基础关键词 = "AI医疗"
扩展策略 = [
    f"{基础关键词} 技术",
    f"{基础关键词} 市场", 
    f"{基础关键词} 政策",
    f"{基础关键词} 2025 趋势"
]
```

#### 源质量评估
```yaml
源质量分级:
  Tier 1 (权重1.0): 学术期刊、政府报告、官方统计
  Tier 2 (权重0.8): 行业报告、权威媒体、专家博客
  Tier 3 (权重0.5): 普通新闻、公司网站
  Tier 4 (权重0.3): 社交媒体、匿名来源
```

### 4. 质量检查清单

**研究前检查**：
- [ ] 研究问题是否清晰明确？
- [ ] 研究范围是否合理界定？
- [ ] 搜索策略是否设计完善？

**研究中检查**：
- [ ] Tier 1来源是否≥30%？
- [ ] 关键主张是否有≥2个独立源支持？
- [ ] 矛盾信息是否妥善处理？

**研究后检查**：
- [ ] 分析框架是否与研究问题匹配？
- [ ] 建议是否具体、可执行？
- [ ] 报告结构是否符合模板规范？

### 5. 常见场景配置

#### 场景1: 技术趋势研究
```yaml
研究模式: 深度推理模式
输出模板: technical.md
搜索轮次: 4-5轮
源质量要求: Tier 1≥40%
分析框架: 技术采纳生命周期2.0 + 技术成熟度曲线
```

#### 场景2: 市场进入分析  
```yaml
研究模式: 并行执行模式
输出模板: executive.md
搜索轮次: 3轮
源质量要求: Tier 1≥30%, Tier 2≥50%
分析框架: PESTEL+ + 波特五力+ + SWOT 2.0
```

#### 场景3: 快速决策支持
```yaml
研究模式: 快速模式
输出模板: quick.md
搜索轮次: 1-2轮
源质量要求: Tier 2+来源为主
分析框架: 选项对比矩阵 + 风险评估
```

### 6. 故障排除速查

| 问题 | 快速解决方案 |
|------|--------------|
| 搜索无结果 | 扩展关键词，调整搜索范围，使用同义词 |
| 源质量低 | 使用高级搜索语法，指定权威域名 |
| 分析困难 | 选择合适的分析框架，分步骤分析 |
| 时间不足 | 降低研究深度，使用快速模板，增加并行度 |
| 结论模糊 | 应用多视角分析，加强证据链构建 |
