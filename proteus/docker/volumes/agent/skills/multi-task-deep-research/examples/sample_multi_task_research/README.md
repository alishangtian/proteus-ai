# 示例: AI教育技术多任务深度研究

## 项目概述
这是一个展示多任务深度研究技能应用的示例项目。项目研究"2025年AI在教育技术中的应用趋势"，包含4个子任务，展示完整的多任务研究流程。

## 研究目标
深度研究2025年AI在教育技术中的应用趋势，包括技术发展、市场应用、政策环境和挑战机遇四个维度，为教育科技公司提供战略决策支持。

## 子任务分解

### 1. 技术发展趋势
研究AI在教育技术中的最新技术发展，包括自适应学习系统、智能辅导、学习分析等技术方向。

### 2. 市场应用分析
分析AI教育技术的市场规模、主要玩家、应用场景和商业模式。

### 3. 政策环境影响
研究各国对AI教育的政策支持、监管框架和伦理指导原则。

### 4. 挑战与机遇
识别AI教育技术发展面临的主要挑战和未来机遇。

## 协调模式
采用**完全并行模式**，四个子任务同时启动，独立执行，最后整合。

## 时间规划
- **阶段1: 规划与分解**: 1天
- **阶段2: 子任务执行**: 3天
- **阶段3: 结果整合**: 1天
- **总计**: 5天

## 质量要求
- 所有子任务质量评分 ≥ 80/100
- 整合深度评分 ≥ 85/100
- 最终报告用户满意度 ≥ 90%
